BioUtils Quickstart

library(BioUtils)

BioUtils user manual

This user manual showcases the functions contained within the BioUtils package.

ID Converter

The ID Converter (id_converter) automatically converts supplied Entrez and Ensembl gene IDs to HGNC gene symbols.

id_converter(ids = c(7157, "ENSG00000118007", "ENSG00000143799"))
#> $`7157`
#> [1] "TP53"
#> 
#> $ENSG00000118007
#> [1] "STAG1"
#> 
#> $ENSG00000143799
#> [1] "PARP1"

Find SNVs

Single nucleotide variants (SNVs) are DNA sequence variations that arise when a single base is altered. The find_snvs function provides the capability to identify SNVs between cancer and normal sequences.

find_snvs(
  cancer = "ATGCGCTA",
  normal = "ATGCTCTT"
)
#>   position cancer normal
#> 1        5      G      T
#> 2        8      A      T

Transcribe

DNA is transcribed into RNA. The transcribe function finds the RNA sequence that will arise from a given DNA sequence.

transcribe(sequence = "AAAGTCGAGGTGTAGATCAAACCC")
#> [1] "UUUCAGCUCCACAUCUAGUUUGGG"

Translate

RNA is then translated into protein. The translate function finds the protein sequence that will arise from the supplied RNA sequence. Note that the RNA sequence length must be evenly divisible by 3 because 3-base codons are used in the function.

translate(sequence = "UUUCAGCUCCACAUCUAGUUUGGG")
#> [1] "FQLHI*FG"

Protein variants

The protein_variant function takes a cancer and normal DNA sequence, computes the resulting protein sequences, and then returns any differences between them.

protein_variant(
  cancer = "AAAGTGGAGGTGTAGATCAAACCC",
  normal = "AAAGTCGAGGTGTAGATGAAACCC"
)
#>   codon_number cancer normal
#> 1            2      H      Q
#> 2            6      *      Y

Find Nonsense Variants

Nonsense variants arise when an SNV causes a premature STOP codon within a sequence. The find_nonsense function scans for nonsense variants within cancer and normal sequences supplied on a per-gene basis. See an example of the input data below:

sequences <- data.frame(
  gene_id = c("ENSG00000147889", 8243, 675),
  cancer = c(
    "AAAGTGGAGGTGTAGATCAAACCC",
    "CATATCCTGATCGGCCTGATCGGGAGG",
    "AGGGCTTTTACCCAGCATTGA"
  ),
  normal = c(
    "AAAGTCGAGGTGTAGATGAAACCC",
    "CATAGCCTGATCGGCCTGAGCGGGAGG",
    "AGGGCTTTTACCCAGGATTGA"
  )
)
sequences
#>           gene_id                      cancer                      normal
#> 1 ENSG00000147889    AAAGTGGAGGTGTAGATCAAACCC    AAAGTCGAGGTGTAGATGAAACCC
#> 2            8243 CATATCCTGATCGGCCTGATCGGGAGG CATAGCCTGATCGGCCTGAGCGGGAGG
#> 3             675       AGGGCTTTTACCCAGCATTGA       AGGGCTTTTACCCAGGATTGA

The result of running find_nonsense on these sequences:

find_nonsense(sequences = sequences)
#>            gene_id symbol codon_number cancer normal
#> 2  ENSG00000147889 CDKN2A            6      *      Y
#> 1             8243  SMC1A            2      *      S
#> 21            8243  SMC1A            7      *      S
